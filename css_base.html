부모태그에 적용된 스타일은 자식태그까지 영향을 끼친다.

스타일의 우선순위는 아래와 같다.

  1. 인라인 스타일
  2. 내부 스타일
  3. 외부 스타일
  4. 디폴트 스타일

네가지 방식의 스타일이 모두 적용되는 것을 스타일 합치기 라고 하고
그 중 동일 태그의 동일 속성이 우선순위에 따라 적용되는 것을 스타일 오버라이딩이라 한다.



제목 태그 h1

부제목 태그 h2

여러 태그에 같은 스타일을 주고 싶을 때는 콤마(,)를 이용하여 나열한다.



class 속성을 가진 태그에 스타일을 주고 싶을 때는 해당 class 속성 이름 앞에 점(.)을 붙인다.

id 속성을 가진 태그에 스타일을 주고 싶을 때는 해당 id 속성 이름 앞에 샾(#)을 붙인다.
