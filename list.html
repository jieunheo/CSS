list

데이터를 나열하여 보여주거나, 목차 및 메뉴를 만들어 관련 링크 모음을 만드는 등 웹 페이지에서 여러가지로 활용된다.
리스트를 꾸미는 기본적인 css 프로퍼티는 다음과 같다.

  • list-style-type: 아이템 마커 타입 지정
  • list-style-image: 아이템 마커 이미지 지정
  • list-style-position: 아이템 마커의 출력 위치 지정
  • list-style: 앞의 3개 프로퍼티를 한 번에 지정하는 단축 프로퍼티
